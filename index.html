The Calimero Project

A free KNX network library.

View on GitHub

Calimero is a Java library offering client access to KNX networks and basic KNX server functionality.

The library is designed to operate on embedded systems. It comes with minimum platform requirements for Java 2 Micro-Edition CDC Foundation Profile (J2ME CDC FP), and is compatible with the popular Java Standard-Edition (J2SE) environments commonly deployed on home computers.

The library's objectives are to offer a clean and lean interface to features including

  • KNXnet/IP tunneling, routing, local device management, and bus monitoring
  • FT1.2 serial access
  • Process communication
  • KNX property access
  • Management procedures
  • Datapoint type and property type translation
  • Network buffering for state- and command-based datapoint values
  • A protocol API resembling OSI layers 2 to 7
